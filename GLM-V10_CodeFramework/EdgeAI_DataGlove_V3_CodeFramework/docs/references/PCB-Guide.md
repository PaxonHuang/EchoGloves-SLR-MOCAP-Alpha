# PCB 布局与布线指南

## 1. 总体布局 (Placement)

### 1.1 主板 (腕部/手背)
- **ESP32-S3**: 放置于手背中心，天线朝外以减少金属干扰。
- **BNO085**: 必须水平放置于手背正中央，轴向与手臂对齐，以确保腕部 Roll/Pitch/Yaw 的准确性。
- **TCA9548A**: 靠近 ESP32-S3 放置，缩短主 I2C 总线长度。

### 1.2 传感器模块 (MCP 关节)
- **TMAG5273**: 每个 MCP 关节（指根）上方放置一个。
- **磁铁安装**: 磁铁安装在近节指骨（Knuckle 之后的第一个指节）。当手指弯曲时，磁铁相对于传感器的距离和角度发生变化。

## 2. 布线规范 (Routing)

### 2.1 I2C 总线
- **上拉电阻**: 在 TCA9548A 的主总线和每个子通道（CH0-CH4）上均需放置 4.7kΩ 上拉电阻。
- **信号完整性**: I2C 走线尽量成对（SDA/SCL）平行，远离电源开关噪声。
- **柔性连接**: 从手背到手指的连接建议使用 30AWG 超软硅胶线或 FPC 柔性电路板，以承受频繁弯曲。

### 2.2 电源设计
- **去耦电容**: 在每个传感器（TMAG5273, BNO085）的 VCC 引脚处放置 0.1uF 去耦电容。
- **地平面**: 保持完整的大面积地平面，减少电磁干扰对霍尔传感器的影响。

## 3. 接口预留
- **ADC 接口**: 在主板边缘预留 5 个焊盘（GPIO 4, 5, 6, 7, 15），用于连接未来的柔性传感器（Flex Sensors）。
- **调试接口**: 预留 UART (TX0/RX0) 焊盘用于固件烧录和日志查看。
