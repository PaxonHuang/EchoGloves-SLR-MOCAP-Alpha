# Hardware-Spec: 传感器布局与电路规范

## 1. 传感器选型理由
- **TMAG5273A1**: 线性 3D 霍尔，支持 I2C 触发模式，比传统柔性传感器寿命长、线性度高。
- **BNO085**: 内置 SH-2 传感器融合算法，输出 Game Rotation Vector（无磁场干扰），极大减轻 ESP32-S3 负担。

## 2. 硬件布局 (Layout)
- **MCP 关节**: 5x TMAG5273 放置于指根背侧。
- **磁铁**: N52 强磁固定于近节指骨，确保弯曲时磁场变化在传感器量程（±40mT）内。
- **手背中心**: BNO085 必须与手掌平面平行，轴向对齐。
- **多路复用**: TCA9548A 解决 5 个相同地址传感器的冲突。

## 3. 信号完整性 (SI)
- **I2C 速率**: 强制 400kHz (Fast Mode)。
- **上拉电阻**: 主总线 2.2kΩ，子通道 4.7kΩ。
- **滤波**: 每个传感器 VCC 必须配备 0.1uF + 10uF 电容。

## 4. 预留扩展
- 预留 GPIO 4, 5, 6, 7, 15 为 ADC 接口，支持未来 L3 阶段的 PIP/DIP 关节柔性传感器接入。
