## 2 I2C 总线接线

### 2.1 主总线 (ESP32-S3 → TCA9548A)

```
ESP32-S3-DevKitC-1                        TCA9548A Breakout
┌──────────────────┐                     ┌─────────────────┐
│                  │                     │                 │
│   3.3V  ─────────┼──┬──────────────────┼── VCC           │
│                  │  │                  │                 │
│   GND   ─────────┼──┼──────────────────┼── GND           │
│                  │  │                  │                 │
│   GPIO8 ─────────┼──┼──────┬───────────┼── SDA           │
│   (SDA)          │  │      │           │                 │
│                  │  │    ┌─┴─┐         │                 │
│   GPIO9 ─────────┼──┼────┼───┼─────────┼── SCL           │
│   (SCL)          │  │    │   │         │                 │
│                  │  │    │   │         │    A0 ── GND    │
└──────────────────┘  │    │   │         │    A1 ── GND    │
                      │    │   │         │    A2 ── GND    │
                      │    │   │         │    /RST ─ 3.3V  │
                      │    │   │         └─────────────────┘
                      │    │   │
           2.2kΩ ─────┘    │   └──── 2.2kΩ
           (SDA 上拉)      │        (SCL 上拉)
                           │
                    都连接到 3.3V
```

> **⚠️ 关键**: TCA9548A 的 A0/A1/A2 全部接 GND → I2C 地址 = 0x70。
> `/RST` 引脚接 3.3V (不使用外部复位)。主总线上拉 2.2kΩ，确保 400kHz Fast Mode 信号完整性。

│  SD3 ───────────┼──────────┼── SDA    TMAG5273 #3 (无名)  │
│  SC3 ───────────┼──────────┼── SCL    addr = 0x22        │

每个子通道 SDA/SCL 都需要 4.7kΩ 上拉到 3.3V

## 3 逐传感器接线详表

### 3.1 TMAG5273 ×5 (手指霍尔传感器)

| 手指 | TCA9548A 通道 | I2C 地址 | VCC | GND | 安装位置 |
|------|-------------|----------|-----|-----|---------|
| 拇指 (Thumb) | Ch0 | 0x22 | 3.3V | GND | 拇指 PIP 关节背侧 |
| 食指 (Index) | Ch1 | 0x22 | 3.3V | GND | 食指 PIP 关节背侧 |
| 中指 (Middle) | Ch2 | 0x22 | 3.3V | GND | 中指 PIP 关节背侧 |
| 无名指 (Ring) | Ch3 | 0x22 | 3.3V | GND | 无名指 PIP 关节背侧 |
| 小指 (Pinky) | Ch4 | 0x22 | 3.3V | GND | 小指 PIP 关节背侧 |

> **复用原理**: 5 个 TMAG5273 的 I2C 地址完全相同 (0x22)，依靠 TCA9548A 通道选择分时访问。每次读取前先 `selectChannel(ch)`，读完后 `disableAll()`。

### 3.2 BNO085 (腕部 IMU)

| 设备 | TCA9548A 通道 | I2C 地址 | VCC | GND | INT | 安装位置 |
|------|-------------|----------|-----|-----|-----|---------|
| BNO085 | Ch5 | 0x4A | 3.3V | GND | GPIO21 | 腕部背侧 |

### 3.3 Flex 传感器 (预留，V3.1 启用)

| 手指 | GPIO | 方向 | 用途 |
|------|------|------|------|
| 拇指 | GPIO 4 | 输入 (ADC1) | Flex Sensor 电压分压读取 |
| 食指 | GPIO 5 | 输入 (ADC1) | 同上 |
| 中指 | GPIO 6 | 输入 (ADC1) | 同上 |
| 无名指 | GPIO 7 | 输入 (ADC1) | 同上 |
| 小指 | GPIO 15 | 输入 (ADC1) | 同上 |

> V3.0 仅将引脚配置为 INPUT (低功耗)，FlexManager 返回全零值。

### 6.2 安装细节

```
      传感器在 PIP 关节背侧的安装 (侧视图)

                     TMAG5273 Breakout
                    ┌────────┐
      近端           │  ●  ●  │           远端
  (Proximal)  ──────│────────│──────  (Distal)
       │             │  PCB   │            │
       │             └────────┘            │
       │     ════════════════════════      │
       │         磁铁 (钕磁铁)               │
       │     ════════════════════════      │
  ┌────┴──────────────────────────────┐
  │         PIP 关节 (侧视)            │
  │    背侧 (Dorsal) ── 传感器在上     │
  │    掌侧 (Palmar) ── 磁铁在下       │
  └───────────────────────────────────┘

  工作原理:
    手指伸直 → 磁铁远离传感器 → 磁场弱 → 输出接近 0
    手指弯曲 → 磁铁靠近传感器 → 磁场强 → 输出增大
    TMAG5273 测量 3D 磁场向量 (X/Y/Z)
    通过三轴分量变化计算弯曲角度
```

### 6.3 磁铁规格建议

| 参数 | 推荐值 |
|------|--------|
| 材质 | N52 钕磁铁 |
| 尺寸 | 3mm 直径 × 1mm 厚 (圆片) 或 3×3×1mm (方块) |
| 磁极方向 | 轴向充磁 (厚度方向) |
| 安装位置 | 手指掌侧 (PIP 关节), 与背侧 TMAG5273 正对 |
| 固定方式 | 缝入手套内层 或 医用胶带粘贴 |

> **关键**: 磁铁与 TMAG5273 的间距应控制在 5-15mm 之间。太远则信号太弱、信噪比低；太近则可能超出 ±40mT 量程、产生削波。

## 7 焊接顺序与注意事项

### 7.1 推荐焊接顺序

```
步骤 1: 电源总线
  ├── 将 3.3V 和 GND 从 ESP32-S3 连接到 TCA9548A
  ├── 从 TCA9548A 引出 3.3V/GND 分支线 (6组)
  └── 验证: 用万用表测量各端点 3.3V ±0.1V

步骤 2: 主 I2C 总线
  ├── 焊接 SDA (GPIO8) → TCA9548A SDA
  ├── 焊接 SCL (GPIO9) → TCA9548A SCL
  ├── 焊接 2.2kΩ 上拉电阻 (SDA→3.3V, SCL→3.3V)
  └── 验证: 上电后用 I2C Scanner 扫描 0x70 (TCA9548A)

步骤 3: 子通道 I2C 总线 (逐通道)
  ├── Ch0: TCA9548A SD0/SC0 → TMAG5273 #0 (拇指)
  │   └── 焊接 4.7kΩ 上拉 (SD0→3.3V, SC0→3.3V)
  ├── Ch1: TCA9548A SD1/SC1 → TMAG5273 #1 (食指) + 4.7kΩ
  ├── Ch2: TCA9548A SD2/SC2 → TMAG5273 #2 (中指) + 4.7kΩ
  ├── Ch3: TCA9548A SD3/SC3 → TMAG5273 #3 (无名指) + 4.7kΩ
  ├── Ch4: TCA9548A SD4/SC4 → TMAG5273 #4 (小指) + 4.7kΩ
  └── Ch5: TCA9548A SD5/SC5 → BNO085 + 4.7kΩ

步骤 4: BNO085 中断线
  ├── 焊接 BNO085 INT → ESP32-S3 GPIO21
  └── 验证: 上电后检查 BNO085 begin() 是否成功

步骤 5: 线缆管理
  ├── 用热缩管包裹每个焊点
  ├── 用尼龙扎带将排线沿手套固定
  ├── 线缆留适当余量 (手指弯曲时不应拉扯焊点)
  └── 关键: 所有焊接点需要应力释放 (strain relief)
```

### 7.2 焊接注意事项

```
⚠️  ESD 防护        ─── 焊接前触摸接地物体, 或使用防静电腕带
⚠️  温度控制        ─── 烙铁温度 300-350°C, 每个焊点 <3 秒
⚠️  上拉电阻        ─── 主总线 2.2kΩ, 子通道 4.7kΩ。不可混用!
⚠️  线长限制        ─── I2C 走线总长 <30cm (400kHz 下最大容性负载 400pF)
⚠️  不要热插拔      ─── 传感器通电时不要插拔 I2C 连接器
⚠️  BNO085 方向     ─── IMU 安装时注意芯片丝印方向，确保坐标系对齐
⚠️  磁铁极性        ─── 5 个手指的磁铁 N 极方向必须一致
⚠️  共地            ─── 所有传感器和 ESP32-S3 必须共地, 否则 I2C 通信失败
```

## 9 常见故障排查

### 9.1 上电无反应 / 串口无输出

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 无任何输出 | 电池耗尽 / USB 未连接 | 充电或更换 USB 线缆 |
| 串口乱码 | 波特率不匹配 | 确认 `monitor_speed = 115200` |
| 反复重启 | PSRAM 未使能 | 检查 GPIO 33 跳线 (N16R8 默认使能) |

### 9.2 传感器初始化失败

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| `TCA9548A not found` | I2C 接线错误 / 地址不对 | 检查 SDA/SCL 连接，A0/A1/A2 是否接 GND |
| `Hall sensors: 0/5` | 子通道上拉缺失 | 检查每个通道 4.7kΩ 上拉电阻 |
| `Hall sensors: 3/5` | 个别传感器焊接问题 | 逐一排查：哪个通道失败 → 重焊该通道 |
| `BNO085 begin() failed` | Ch5 未选择 / INT 未连接 | 检查 Ch5 上拉，确认 BNO085 的 INT 引脚连到 GPIO21 |
| `Timeout on mux chX` | 传感器 NACK / 总线锁死 | 检查传感器 VCC/GND，确认传感器未损坏 |

### 9.3 数据异常

| 现象 | 可能原因 | 解决方法 |
|------|---------|---------|
| 所有通道数据相同 | TCA9548A 通道未切换 | 检查 selectChannel() 逻辑 |
| Hall 数据全部为 0 | 磁铁未安装 / 距离太远 | 调整磁铁位置 (<15mm) |
| Hall 数据跑到 ±40mT | 磁铁太近 / 超出量程 | 增大磁铁间距或换小磁铁 |
| 欧拉角漂移 | BNO085 未完成校准 | 上电后静止 2 秒完成自动校准 |
| I2C 间歇性失败 | 走线太长 / 上拉太弱 | 缩短排线 <30cm，确认上拉 2.2kΩ |

### 9.4 快速诊断命令

```cpp
// 在 setup() 中添加 I2C 扫描代码以诊断硬件:

#include <Wire.h>
void i2cScanner() {
    Serial.println("I2C Scanner:");
    for (uint8_t addr = 1; addr < 127; addr++) {
        Wire.beginTransmission(addr);
        if (Wire.endTransmission() == 0) {
            Serial.printf("  Device found at 0x%02X\n", addr);
        }
    }
}

// 期望结果:
//   0x22 — TMAG5273 (需先 TCA9548A selectChannel)
//   0x4A — BNO085 (需先 TCA9548A selectChannel(5))
//   0x70 — TCA9548A (始终可见)
```
